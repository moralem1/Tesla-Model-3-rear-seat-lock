# Summary

Locking mechanism for Tesla Model 3 rear seats. There are 2 versions of the "rocker arm", one for a simple toggle, and one for the extended linkage. 

Video of operation, assembly, and installation: https://youtu.be/8tt10vhaUjw

Edit: Sorry if mixing all the files together was confusing...here is the breakdown of parts (I also added a zip file for each lock type if that's easier):

For each type of lock, you will need:
1. base.stl
2. bolt.stl
3. rocker_arm_cap.stl

For the simple lock, you also need:
4. rocker_arm_no_linkage.stl

For the extended arm lock, you also need:
4. rocker_arm_for_linkage.stl
5. extension_arm_link_end.stl
6. extension_arm_lever_end.stl
7. extension_arm_guide.stl
